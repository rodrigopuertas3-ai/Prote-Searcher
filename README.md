# ProteSearch: 
Simply enter the protein name/ID or the link from *www.uniprot.org/uniprotkb*, and **ProteSearch** will extract the following information into an Excel file:
- Sequence ID
- EMBL/GenBank ID(s)
- Chromoprotein Name
- Sequence (aa)
- Sequence Length (aa)
- pI (theoretical)
- Organism of Origin
- Absorbance (nm)
- Emission (nm)
- References (up to 3)
- Molecular Weight (Da)
- UniProt Link
